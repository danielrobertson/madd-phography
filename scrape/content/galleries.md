# Galleries / Portfolio  (/clients/)

Client gallery collection. Galleries are filterable by tag:
- `/clients/?tag=graduation`
- `/clients/?tag=wedding`

Each entry is a client session shown as a cover photo linking into a private Pixieset
gallery (individual client deliverables). The 13 public **cover images** for these
galleries were downloaded to `images/galleries/`.

The curated marketing/portfolio imagery (used across Home, Services, About, Reviews)
lives in the `elementfield` image set and was downloaded at full 2500px resolution
into the respective page folders.
